# Hakruo

Manga application for reading and downloading manga.

## Requirements

[Node.js v10^](https://nodejs.org/en/)

[Yarn v1.13.0](https://yarnpkg.com/en/)

## Installation

```
git clone https://github.com/Shinudesu/Hakuro.git
yarn install
```

## Running

For development:

```
yarn electron-dev
```

For production:

```
yarn electron-release
```
